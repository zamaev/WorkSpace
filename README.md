# WorkSpace

Локальный однопользовательский планировщик: дерево задач + недельный вид.
Дизайн — визуальный язык проекта space. Спека: `docs/superpowers/specs/`.

## Запуск

```sh
docker compose up --build -d   # приложение на http://localhost:8787
```

Данные — SQLite-файл в `./data/` (переживает пересборку и перезапуск).

## Разработка

```sh
make dev-api    # go run, порт 8787
make dev-front  # vite dev, проксирует /api на 8787
make test       # go vet + go test, tsc + vitest
```

## Как пользоваться

- **Проекты** (`/projects`): слева список проектов (цвет, порядок drag'ом,
  переименование кликом по имени активного, палитра — кликом по полоске
  цвета, удаление каскадом), справа — дерево задач выбранного проекта.
  У каждой задачи слева полоска цвета проекта. Дата — через чип/◷ (меню:
  сегодня · завтра · дни недели · календарь). Drag по дереву: середина
  строки — внутрь, верх/низ — сиблингом до/после.
- **Каскад done**: отметил задачу — полностью сделанные ветки закрываются
  вверх сами; снял отметку или добавил задачу — предки открываются.
- **Дедлайны**: у задачи кроме плана («в какой день делаю») есть дедлайн
  «⚑ до какого числа» — второй чип в дереве и на карточке недели; у
  проекта — даты старта и дедлайна (для Ганта). Плашка «Просрочено» в
  неделе делится на «Сорван дедлайн» и «Не сделано в запланированный
  день».
- **Неделя** (`/week`): 7 колонок Пн–Вс с цветными полосками проектов,
  drag между днями и внутри дня, быстрый ввод в ячейке (точка слева —
  выбор проекта), листание ◂ ▸.
- **Гант** (`/gantt`): проекты полосами на шкале дней (drag середины —
  сдвиг, края — старт/дедлайн отдельно; «＋ даты» для проекта без дат),
  раскрытие проекта показывает задачи (полоса план→дедлайн, ромб — одна
  дата, done приглушены), линия «сегодня», выходные подсвечены.
- Хоткеи: `1` — проекты, `2` — неделя, `3` — Гант, `T` — текущая неделя.
  Тумблер темы в шапке (тёмная — дефолт).
