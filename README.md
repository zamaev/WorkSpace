# WorkSpace

Локальный однопользовательский планировщик: дерево задач + недельный вид.
Дизайн — визуальный язык проекта space. Спека: `docs/superpowers/specs/`.

## Запуск

```sh
docker compose up --build -d   # приложение на http://localhost:8787
```

Данные — SQLite-файл в `./data/` (переживает пересборку и перезапуск).

## Разработка

```sh
make dev-api    # go run, порт 8787
make dev-front  # vite dev, проксирует /api на 8787
make test       # go vet + go test, tsc + vitest
```

## Как пользоваться

- **Дерево** (`/`): ветки любой вложенности; чекбокс done независим от детей,
  рядом счётчик «сделано/всего» по прямым детям. Дата — через чип/◷ (меню:
  сегодня · завтра · дни недели · календарь) или перетаскиванием узла на
  полоску недели внизу. Drag по дереву: середина строки — внутрь, верх/низ —
  сиблингом до/после.
- **Неделя** (`/week`): 7 колонок Пн–Вс, drag между днями и внутри дня,
  быстрый ввод в ячейке (создаёт корневую задачу с датой), плашка
  «Просрочено» с действиями «на сегодня»/«снять дату», листание ◂ ▸.
- Хоткеи: `1` — дерево, `2` — неделя, `T` — текущая неделя. Тумблер темы
  в шапке (тёмная — дефолт).
