# WorkSpace

Локальный однопользовательский планировщик: дерево задач + недельный вид.
Дизайн — визуальный язык проекта space. Спека: `docs/superpowers/specs/`.

## Запуск

```sh
docker compose up --build -d   # приложение на http://localhost:8787
```

Данные — SQLite-файл в `./data/` (переживает пересборку и перезапуск).

## Разработка

```sh
make dev-api    # go run, порт 8787
make dev-front  # vite dev, проксирует /api на 8787
make test       # go vet + go test, tsc + vitest
```

## Как пользоваться

- **Проекты** (`/projects`): слева дерево проектов (вложенность, drag:
  середина — под-проектом, край — сиблингом; цвет — кликом по полоске,
  переименование — кликом по имени активного), справа — дерево задач
  выбранного. Пустой проект удаляется (✕), непустой — только в архив (▣,
  рекурсивно с под-проектами); секция «Архив» внизу, оттуда можно вернуть.
  Задачу можно перетащить на строку проекта — она уедет в его корень.
  Клик по задаче открывает инспектор справа (название, сроки, тип,
  исполнитель, описание, удаление-корзина с подтверждением вторым кликом);
  переименование — двойной клик. Ширины колонок тянутся за разделители.
  Даты — свой календарь (чипы «Сегодня · Завтра · Пн», сетка месяца,
  у плана — режим «по…» для диапазона работы).
  Drag по дереву задач: середина строки — внутрь, верх/низ — сиблингом.
  Отметки done полностью ручные — никаких автокаскадов; дедлайн не может
  быть раньше запланированного дня.
- **Дедлайны**: у задачи кроме плана («в какой день делаю») есть дедлайн
  «⚑ до какого числа» — второй чип в дереве и на карточке недели; у
  проекта — даты старта и дедлайна (для Ганта). Плашка «Просрочено» в
  неделе делится на «Сорван дедлайн» и «Не сделано в запланированный
  день».
- **Неделя** (`/week`): 7 колонок Пн–Вс (тумблер «Сб–Вс» прячет выходные)
  с цветными полосками проектов, drag между днями и внутри дня, быстрый
  ввод в ячейке (точка слева — выбор проекта), листание ◂ ▸, тумблер
  «2 нед» — следующая неделя второй строкой (перетаскивание между
  неделями). Клик по задаче — попап с деталями. Задачи архивных проектов
  скрыты. Многодневная задача (план + «работаю по…») видна в каждом дне
  диапазона: полная карточка в первый день, «продолжения» дальше.
- **Гант** (`/gantt`): иерархия проектов полосами на шкале дней (drag
  середины — сдвиг, края — старт/дедлайн отдельно; «＋ даты» для проекта
  без дат), раскрытие показывает задачи и под-проекты (полоса
  план→дедлайн, ромб — одна дата, done приглушены), линия «сегодня»,
  числа всех дней, пунктирная сетка, выходные подсвечены, тумблер «Архив».
- **Команда** (`/team`): люди с цветом и инициалами + справочник ролей
  во второй колонке (роль назначается человеку чипом). Исполнитель
  задачи — в карточке задачи, можно снять. У проекта — несколько
  участников (стек аватарок в шапке дерева, состав — попапом).
- **Типы** (`/types`): справочник типов задач со смайликами (пресеты или
  любой свой — ⌃⌘Space); в дереве, неделе и на Ганте тип показывается
  смайликом, имя — в тултипе.
- Карточка задачи: одна линия чипов (план · дедлайн · тип · исполнитель),
  меню — маленькие попапы; календарь плана с переключалкой «День |
  Диапазон». Крошки пути — только в попапе недели.
- Дедлайн двойной: мягкий (цель) и жёсткий (крайний срок), попап с
  вкладками «мягкий | жёсткий». Компактный чип показывает ближайший
  рубеж: синеватый до мягкого, янтарный после него, красный после
  жёсткого; на Ганте мягкий — полый кружок, жёсткий — флажок, оба
  таскаются. «Просрочка» в неделе считается по жёсткому.
- ⌘K — поиск по названиям задач и активных проектов с переходом в
  дерево; в «Проектах» ↑/↓ ходят по строкам, инспектор следует.
- Фильтры недели и Ганта (общие): по исполнителю (аватарки) и по типу
  (смайлики); повторный клик снимает фильтр.
- В «Проектах»: тумблер «скрыть ✓» прячет сделанные ветки; при
  перетаскивании задачи внизу появляется полоска-неделя (две недели,
  если в «Неделе» включён тумблер «2 нед») — бросай на день;
  выбранная задача переживает перезагрузку; описание и название
  сохраняются сами через паузу набора.
- Команда и роли сортируются перетаскиванием; порядок отражается во всех
  списках.
- При старте сервер кладёт копию базы в `data/backups/` (последние 10).
- Подсказки: кнопка «?» в шапке — клавиши и жесты.
- Хоткеи: `1` — проекты, `2` — неделя, `3` — Гант, `4` — команда, `5` —
  типы, `T` — текущая неделя. Тумблер темы в шапке (тёмная — дефолт).
