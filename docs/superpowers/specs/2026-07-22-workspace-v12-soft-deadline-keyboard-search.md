# WorkSpace v12 — мягкий/жёсткий дедлайн, клавиатура, Cmd+K, фикс полоски

> Дата: 2026-07-22. Одобрено целиком «как рекомендуешь»: янтарный для
> средней фазы, компактные чипы показывают ближайший рубеж.

## 1. Двойной дедлайн

### Модель (миграция 0008)

- `tasks.soft_due_on TEXT NULL` — мягкий дедлайн. Существующий `due_on`
  остаётся жёстким (смысл данных не меняется).
- Инварианты: `soft_due_on >= scheduled_on`, `due_on >= soft_due_on`
  (каждый — если обе стороны заданы; `due_on >= scheduled_on` уже есть).
  Любая комбинация «только мягкий / только жёсткий / оба» валидна.

### API

- `softDueOn` в GET/POST/PATCH `/api/tasks` (PATCH — Opt, null значим).

### Фазы и цвета (фронт)

`duePhase(soft, hard, today)` → ближайший рубеж + фаза:

- сегодня < мягкого → `soft` (сине-серый), показывается мягкая дата;
- мягкий прошёл, жёсткий впереди → `warn` (янтарь), жёсткая дата;
- жёсткий прошёл → `over` (красный, заливка), жёсткая дата;
- только жёсткий: `soft` до него, `over` после (янтарной фазы нет);
- только мягкий: `soft` до, `warn` после (красной не бывает);
- done-задача: чип нейтральный (фазы не подсвечиваются).

Токены: `--due-soft` (сине-серый), `--due-warn` (янтарь), красный —
существующий `--over`. Классы `chip-due-soft/-warn/-hard`.

### UI

- Карточка задачи: попап дедлайна — переключалка «мягкий | жёсткий»
  (паттерн «День | Диапазон»), календарь + «Снять» для выбранного рубежа.
  Чип: обе даты «Чт 24 · Пт 26», цвет по фазе.
- Дерево: чип — ближайший рубеж (см. фазы), клик — тот же попап.
- Неделя: подпись «до …» — ближайший рубеж и его цвет.
- Плашка просрочки: по жёсткому, как раньше (мягкий не считается срывом).
- Гант: мягкий — полый янтарный кружок, жёсткий — флажок как сейчас;
  пунктирный хвост — до самого позднего заданного рубежа; кружок можно
  тянуть (кламп: план ≤ мягкий ≤ жёсткий).

## 2. Фикс: полоска-неделя не исчезает

Root cause: drop-обработчик строки дерева гасит событие
(`stopPropagation` — легитимно, защита от вложенных зон), а `dragend` не
приходит: React перемонтировал источник drag'а после переноса в
подзадачи. Фикс: DragWeekStrip слушает `drop`/`dragend`/`dragstart` на
window в **capture-фазе** — срабатывает раньше любых stopPropagation.

## 3. Клавиатура в дереве

- ↑/↓ — переход по видимым строкам дерева (свёрнутые ветки и скрытые
  done пропускаются), выбор синхронизирован с инспектором.
- Не срабатывает, когда фокус в input/textarea или открыт модал/попап.
- Упоминание в «?»-шпаргалке.

## 4. Cmd+K — палитра поиска

- ⌘K (и Ctrl+K) в любом виде → оверлей с полем; фильтр по подстроке
  (без регистра) названий задач и активных проектов; до 20 результатов;
  у задачи — крошка проекта; done — приглушены.
- ↑/↓ + Enter или клик: задача → `/projects/{projectId}?focus={id}`,
  проект → `/projects/{id}`. Esc закрывает. Упоминание в шпаргалке.

## 5. Тесты
