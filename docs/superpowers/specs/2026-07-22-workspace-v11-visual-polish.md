# WorkSpace v11 — визуальная полировка

> Дата: 2026-07-22. Прямые поручения пользователя, без развилок.

1. Drag-неделя в «Проектах» показывает 2 недели (второй ряд), если в
   «Неделе» включён тумблер «2 нед»; уважает «Сб–Вс».
2. Флаг дедлайна: символ ⚑ заменён на штриховой SVG-флажок (в чипах);
   в строках дерева флаг у чипа дедлайна убран совсем.
3. Счётчик подзадач «0/2» из строк дерева убран (шум).
4. Цвета чипа дедлайна: всегда красноватый (мягкая заливка/текст),
   при наступлении/просрочке — жёстко красный (заливка --over, белый
   текст). План остаётся акцентным. То же в подписи «до …» карточек.
5. Исполнитель: концепция «пусто = я» удалена. Чип «исполнитель» (как
   «тип»), пункт меню «снять исполнителя»; из фильтров кнопка «я» ушла.
6. Фильтры: активная аватарка — ровное круглое кольцо (box-shadow вместо
   кривого outline); кнопки сброса нет — повторный клик снимает; группы
   «люди» и «типы» разделены отступом и тонкой чертой.
7. Иконка архива проекта: вместо ▣ — SVG-коробка с крышкой.
8. Неделя: попап задачи открывается кликом по любому месту карточки
   (чекбокс/точка проекта — не в счёт).
9. Команда: «Роли» — второй колонкой справа от списка людей.
